getElementsByAttribute( attrib, value )
Return type: DOM NodeList
Returns a NodeList of all the child elements of the element that have the attribute given by the first argument set to the value given by the second argument. If the second argument is '*', the method will match the given attribute set to any value. Note that like most NodeLists (but unlike the NodeList returned by querySelectorAll), the NodeList returned by this method is live.
Note that this method is only available on XUL elements; it is not part of the W3C DOM.
