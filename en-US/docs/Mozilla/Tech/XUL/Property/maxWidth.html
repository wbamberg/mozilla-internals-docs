maxWidth
Type: integer
Gets and sets the value of the maxwidth attribute.

Note: Prior to Gecko 6.0 , you could use this on <input> elements; this was never intended, was a violation of the HTML specification, and has been fixed. Use the size to set the maximum width of input fields.
