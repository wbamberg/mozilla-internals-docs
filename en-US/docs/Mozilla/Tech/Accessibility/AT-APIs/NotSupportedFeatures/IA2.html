List of unsupported features

Below you will find what methods/attributes of IA2 interfaces we do not support.

If you need the following features (if you need something, please contact Aaron Leventhal )

IAccessible2

  • localizedExtendedRole
  • nExtendedStates
  • extendedStates
  • localizedExtendedStates

IAccessibleAction

  • localizedName

IAccessibleRelation

  • localizedRelationType

IAccessibleTable

  • summary
  • modelChange
