Â« Gecko Roles Page

Represents a menu, which presents a list of options from which the user can make a selection to perform an action.

Interfaces

  • nsIAccessNode
  • nsIAccessible
  • nsIAccessibleSelectable

Events

  • EVENT_MENUPOPUPSTART - fired when menu is shown
  • EVENT_MENUPOPUPEND - fired when menu is hidden

Mapped to

  • AT-SPI: ROLE_MENU
  • ATK: ATK_ROLE_MENU
  • UA: NSAccessibilityMenuRole
  • MSAA/IA2: ROLE_SYSTEM_MENUPOPUP

Used by

  • ARIA: menu
  • XUL: < popup />, < menupopup />
