steelIApplication

This content covers features introduced in Thunderbird 3

The

Implemented via XPCOM service for extIApplication : see the instructions on the FUEL (Firefox), STEEL (Thunderbird) and SMILE (SeaMonkey) pages.

Attributes

Attribute Type Description
platformIsMac readonly attribute boolean Returns true if the application is running on a Mac OS X. This is intended for situations where Mac does things differently, e.g. The application doesn't support print preview on Mac because Mac provides a preview option in the print dialog.
platformIsLinux readonly attribute boolean Returns true if the application is running on GNU/Linux or on UNIX New in Thunderbird 20
