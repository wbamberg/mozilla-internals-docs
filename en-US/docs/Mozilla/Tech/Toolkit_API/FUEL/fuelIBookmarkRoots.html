fuelIBookmarkRoots

FUEL is deprecated as of Firefox 40. Consider using the Add-ons SDK instead

Attributes

Attribute Type Description
menu readonly attribute fuelIBookmarkFolder The bookmark menu root.
toolbar readonly attribute fuelIBookmarkFolder The personal toolbar root.
tags readonly attribute fuelIBookmarkFolder The tag root.
unfiled readonly attribute fuelIBookmarkFolder The unfiled bookmarks root.

Examples 

var bookmark = Application.bookmarks.unfiled.addBookmark("Test", testURI);
