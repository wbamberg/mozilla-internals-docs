NS_CStringContainerFinish

« XPCOM API Reference

Summary

The NS_CStringContainerFinish function releases any memory allocated by a nsCStringContainer instance. 

#include "nsStringAPI.h"
 
void NS_CStringContainerFinish(
  nsCStringContainer& aString
);

Parameters

aString
[in] A nsCStringContainer instance that is no longer needed.

History

This function was finalized for Mozilla 1.7. See bug 239123 for details.
