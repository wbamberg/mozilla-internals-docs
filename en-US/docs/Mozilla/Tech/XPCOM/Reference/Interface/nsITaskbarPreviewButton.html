nsITaskbarPreviewButton

widget/public/nsITaskbarPreviewButton.idlScriptable
This interface is used on Microsoft Windows to get access to a window preview's toolbar button properties. These buttons are displayed below the window preview.
1.0
66
Introduced
Gecko 1.9.2
Inherits from: nsISupports Last changed in Gecko 1.9.2 (Firefox 3.6 / Thunderbird 3.1 / Fennec 1.0)

You can't directly instantiate this interface. Instead, you use the nsITaskbarWindowPreview.getButton() to get the objects representing the buttons.

Attributes

Attribute Type Description
disabled boolean If true, the button is disabled. This is not the same as visible, which indicates whether or not the button should be displayed at all.
dismissOnClick boolean If true, the array of previews should be dismissed when the button is clicked. This is false by default.
hasBorder boolean If true, the taskbar should draw a border around this button's image. This is true by default.
image imgIContainer The icon displayed in the button. This is null by default.
tooltip DOMString The string displayed in the button's tooltip. By default, this is an empty string.
visible boolean If true, the button is displayed. If false, the button doesn't participate in the layout of buttons underneath the preview. This is false by default.
