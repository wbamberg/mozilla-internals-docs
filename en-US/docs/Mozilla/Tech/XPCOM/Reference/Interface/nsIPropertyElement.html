nsIPropertyElement

xpcom/ds/nsIPersistentProperties2.idlScriptable
This interface provides access to individual entries within a stringbundle.
Inherits from: nsISupports Last changed in Gecko 1.0

Attributes

Attribute Type Description
key AUTF8String The key used to refer to this property.
value AString The string value stored for this property.
