nsIDOMGeoPositionError

The nsIDOMGeoPositionError interface represents an error that occurred while performing geolocation operations.

dom/interfaces/geolocation/nsIDOMGeoPositionError.idlScriptable
Please add a summary to this article.
  Last changed in Gecko 1.9.2 (Firefox 3.6 / Thunderbird 3.1 / Fennec 1.0)

Inherits from: nsISupports

Attributes

Attribute Type Description
code short Numerical error code; see Error constants for a complete list. Read only.
message DOMString Human readable error message. Read only.

Gecko 1.9.2 note
The message attribute was removed in Gecko 1.9.2 (Firefox 3.6).

Error constants

Constant Value Description
PERMISSION_DENIED 1 The user denied permission to retrieve location data.
POSITION_UNAVAILABLE 2 Unable to access location data.
TIMEOUT 3 A timeout occurred while trying to retrieve location data
