recalculate()

Signals the XForms Processor to perform a full recalculation of the given XForms Model.

Note:Â Script invocation is not necessarily equivalent to performing the recalculate action handler. Though the script is assumed to have modified instance data prior to invoking recalculate, the DOM mutations are not cached. Thus, a full recalculation is necessary to ensure the proper changes are effected throughout the XForms Model.

void recalculate();
Parameters

None.
