Methods

installApplication()

This method installs a web application.

void installApplication(
  in wstring title,
  in wstring uri,
  in wstring iconUri,
  in wstring iconData
);
Parameters
title
The user-friendly name of the application.
uri
The uri of the web application.
iconUri
The uri of icon of the web application.
iconData
A base64 encoded representation of the application's icon.

isApplicationInstalled()

Checks is a web application is already installed.

boolean isApplicationInstalled(
  in wstring uri
);
Parameters
uri
The uri of the web application.
Return value

true if the web application is installed, false if it's not installed.
