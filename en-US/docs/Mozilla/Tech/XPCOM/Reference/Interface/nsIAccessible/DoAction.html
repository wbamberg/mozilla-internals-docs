Summary

This method performs the accessible action at the given zero-based index. Action number 0 is the default action.

 void doAction(
   in PRUint8 aIndex
 );

Parameters

aIndex
[in] The zero-based index.

Exceptions thrown

NS_ERROR_FAILURE
Indicates that the accessible is unattached from the accessible tree.
NS_ERROR_INVALID_ARG
Indicates that the given index is our of range.
