nsIMsgSendLater

The nsIMsgSendLater interface provides functions for managing the unsent folder of a messaging account.

Inherits from: nsIStreamListener

Implemented by: @mozilla.org/messengercompose/sendlater;1. To create an instance, use

var msgSendlater = Components.classes["@mozilla.org/messengercompose/sendlater;1"]
                             .getService(Components.interfaces.nsIMsgSendLater);
