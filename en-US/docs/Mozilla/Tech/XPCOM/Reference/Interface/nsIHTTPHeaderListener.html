nsIHTTPHeaderListener

modules/plugin/base/public/nsIHTTPHeaderListener.idlScriptable
This interface allows plugin authors to access HTTP Response headers after issuing an nsIPluginHost.GetURL or nsIPluginHost.PostURL call.
Inherits from: nsISupports Last changed in Gecko 1.9 (Firefox 3)

Note: The plugin author must provide an instance to {GetURL,PostURL}() that implements both nsIPluginStreamListener and nsIHTTPHeaderListener. This instance is passed in through {GetURL,PostURL}()'s streamListener parameter. The browser will then QI this streamListener to see if it implements nsIHTTPHeaderListener.
