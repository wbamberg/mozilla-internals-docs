nsITXTToHTMLConv

netwerk/streamconv/public/nsITXTToHTMLConv.idlScriptable
This interface allows you to modify the conversion from plain text to HTML.
Inherits from: nsIStreamConverter Last changed in Gecko 1.8 (Firefox 1.5 / Thunderbird 1.5 / SeaMonkey 1.0)
Implemented by "@mozilla.org/streamconv;1?from=text/plain&to=text/html". You should normally obtain an instance via the nsIStreamConverterService interface.Â
