nsIGlobalHistory2

docshell/base/nsIGlobalHistory2.idl Scriptable
This interface provides information about global history to Gecko. It was split off from nsIGlobalHistory during the transition to toolkit interfaces.
Inherits from: nsISupports Last changed in Gecko 1.7

This interface replaces and deprecates nsIGlobalHistory

addURI()

Add a URI to global history.

Note: Docshell will not filter out URI schemes like chrome: , data: , about: and view-source: . Embedders should consider filtering out these schemes and others, for example mailbox: , for the mainÂ  nsIURI and the referrer.

Parameters
aURI
The nsIURI of the page being added.

Whether the URI was redirected to another location; this is true for the original URI which is redirected.

aReferrer
