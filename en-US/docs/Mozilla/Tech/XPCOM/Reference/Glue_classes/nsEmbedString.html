nsEmbedString

« XPCOM API Reference

Summary

The nsEmbedString concrete class provides a way to construct a nsAString object that allocates null-terminated storage.

  #include "nsEmbedString.h"
  // Or: #include "nsStringGlue.h" if inside Gecko

  class nsEmbedString : public nsAString { ... };

Methods

nsEmbedString
Constructors for nsEmbedString.
get
The get function returns a const pointer to the string's null-terminated, internal buffer.

Operators

operator=
This operator is a shortcut for the inherited nsAString::Assign family of functions.

Remarks

The methods defined on nsEmbedString are implemented as inline wrappers around the XPCOM string functions, prefixed with NS_String. See, for example, NS_StringContainerInit.
