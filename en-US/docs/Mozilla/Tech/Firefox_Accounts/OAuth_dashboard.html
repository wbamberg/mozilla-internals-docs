Firefox Accounts OAuth Dashboard

FxA OAuth Credential Management Dashboard is a web application that lets you provision Firefox Accounts OAuth credentials.

Note: You must have an @mozilla.com email to access this application. You might need to register a new test account on any of the available environments listed below.

When creating new clients, make sure to check the Whitelisted Client checkbox. Otherwise the client will not be able to authenticate.

Available environments

  • https://oauth-latest.dev.lcip.org/console
  • https://oauth-stable.dev.lcip.org/console
